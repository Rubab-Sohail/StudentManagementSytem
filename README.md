# Student Management System

A Web Engineering semester project built using HTML, CSS, Bootstrap, JavaScript, jQuery, PHP, and MySQL.

## Features

- Student registration form with front-end validation
- Add, view, update, and delete student records (CRUD)
- Responsive design using Bootstrap 5
- jQuery hide/show animation for the registration form
- PHP backend connected to a MySQL database

## Project Structure

```
StudentManagementSystem/
├── frontend/           # HTML, CSS, JS (Part 2)
│   ├── index.html
│   ├── style.css
│   └── script.js
├── backend/            # PHP + MySQL (Part 3)
│   ├── index.php
│   ├── insert.php
│   ├── display.php
│   ├── update.php
│   ├── delete.php
│   ├── db.php
│   └── style.css
└── student_management.sql
```

## Technologies Used

- HTML5 & CSS3
- Bootstrap 5
- JavaScript & jQuery
- PHP
- MySQL

## How to Run

1. Install [XAMPP](https://www.apachefriends.org/)
2. Copy the `StudentManagementSystem` folder into `htdocs`
3. Start Apache and MySQL from the XAMPP Control Panel
4. Open phpMyAdmin and import `student_management.sql`
5. Open your browser and go to `http://localhost/StudentManagementSystem/backend/`
